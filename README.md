# 📰 Blog Web
A modern and responsive full-stack blogging platform where users can explore, read, and manage blog posts through a clean and intuitive interface. The application focuses on performance, user experience, and scalability using modern web technologies.

# 🚀 Project Overview

Blog Web is a responsive blogging platform that enables users to browse, read, and interact with blog content through a clean and modern interface. Built with Next.js and TypeScript, it delivers a fast, scalable, and user-friendly experience.

# ⚙️ Getting Started

## Clone Repository

```bash
git clone https://github.com/Mahabub-Alom-Sujon/blog_web.git
```

## Navigate to Project

```bash
cd blog_web
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open

```
http://localhost:3000
